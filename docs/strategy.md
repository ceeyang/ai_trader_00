# 交易策略详解与对比 (Trading Strategy Analysis)

本文档详细对比了两种核心交易策略：**策略 A (主流币再平衡)** 与 **策略 B (山寨币搏金狗)**。

---

## 策略 A: 主流币再平衡 (Mainstream Rebalancing)
**核心哲学**: 均值回归 (Mean Reversion) — "别人恐惧我贪婪"。

### 1. 策略逻辑
*   **标的**: 市值前 10 的主流确定性资产 (BTC, ETH, BNB, SOL 等)。
*   **机制**: 设定目标仓位比例 (如 20%)。
    *   **下跌时**: 仓位价值缩水 -> 比例降低 -> **买入补仓** (Buy the Dip)。
    *   **上涨时**: 仓位价值膨胀 -> 比例升高 -> **卖出止盈** (Sell the Rip)。
*   **对待下跌**: 视为**机会**，通过补仓降低持仓均价。

### 2. 优劣势分析
| 维度 | 优势 (Pros) | 劣势 (Cons) |
| :--- | :--- | :--- |
| **收益特性** | **稳健复利**。在震荡市表现极佳，反复收割波动率。 | **收益有顶**。上涨时不断减仓，难以吃到单边大牛市的全部涨幅。 |
| **风险特征** | **低归零风险**。因为标的是主流币，大跌通常是暂时的。 | **被套风险**。单边熊市中会一路补仓，造成短期大额浮亏 (Painful Drawdown)。 |
| **适用资金** | 大资金、追求稳健增值、长期主义者。 | 小资金、追求短期暴富者不适用。 |

### 3. 可行方案配置
*   `MAX_OPEN_POSITIONS`: 5 ~ 10
*   `LEVERAGE`: 1x ~ 3x
*   `BLACKLIST`: 仅排除非主流币 (保留 BTC, ETH)
*   `STOP_LOSS`: 宽止损 (如 30% 全局回撤)

---

## 策略 B: 山寨币优胜劣汰 (Altcoin Moonshot)
**核心哲学**: 趋势动量 (Momentum / Survival of the Fittest) — "截断亏损，让利润奔跑"。

### 1. 策略逻辑
*   **标的**: 40+ 个中盘热门山寨币 (Meme, AI, GameFi, Layer2 等)。
*   **机制**: 广撒网，高分散。
    *   **下跌时**: 单币跌幅达到阈值 (如 -30%) -> **立即止损离场** (Cut Losers)。
    *   **上涨时**: 不动或加仓 -> **持有直到趋势结束** (Ride Winners)。
*   **对待下跌**: 视为**错误**，承认看走眼，保留本金火种。

### 2. 优劣势分析
| 维度 | 优势 (Pros) | 劣势 (Cons) |
| :--- | :--- | :--- |
| **收益特性** | **爆发性强**。只需押中 10% 的百倍币 (10x-100x)，即可覆盖所有磨损并大幅盈利。 | **高磨损 (Churn)**。在行情不明朗时，会频繁触发止损，本金像“钝刀割肉”般减少。 |
| **风险特征** | **断尾求生**。遇到黑天鹅（项目归零）时，只损失单仓位 (如 2.5%)，绝不陪葬。 | **踏空风险 (Whipsaw)**。刚止损离场，价格就反转暴涨，心态容易崩。 |
| **适用资金** | 小资金、风险偏好高、以小博大。 | 心理素质差、厌恶频繁亏损者不适用。 |

### 3. 可行方案配置
*   **分散度**: `MAX_OPEN_POSITIONS = 40` (单仓位仅占 2.5% ~ 5%)
*   **黑名单**: `BLACKLIST = ['BTC', 'ETH', 'BNB', ...]` (强制排除平稳的主流币)
*   **止损**: **必须启用单币硬止损**。
    *   **当前代码需修改**: 将 `Rebalancer` 的补仓逻辑改为：若 `PnL < -30%` -> `Close Position`。
*   **杠杆**: 1x ~ 2x (不宜过高，防止波动误杀)

## 总结与建议

| 市场阶段 | 推荐策略 | 理由 |
| :--- | :--- | :--- |
| **熊市 / 震荡市** | **策略 A (再平衡)** | 山寨币普跌，持有现金或主流币利用波动率赚钱最安全。 |
| **牛市初期 / 中期** | **策略 B (搏金狗)** | 流动性溢出，垃圾币也会飞。此时要敢于拥抱泡沫，利用高赔率博取超额收益。 |

**当前建议**:
鉴于目前处于加密市场周期（可能是牛市初期），且您希望利用杠杆增加收益，**策略 B (小资金试错版)** 值得尝试。建议用小仓位跑策略 B，体验其“优胜劣汰”的逻辑，同时打磨心态。
